About This App
Purpose

A training tool for vintage clothing dealers to sharpen pricing instincts. Browse sold items from Mushroom Vintage (Tokyo) and test your ability to estimate prices in the flashcard game.

Data Source

All items are scraped from vintage-mushroom.net, a high-end vintage store in Tokyo. Only items sold for ¥65,000+ (~$430 USD) are included.

Prices are converted at ¥150 = $1 USD. Descriptions are auto-translated from Japanese using Google Translate.

Tech Stack
  • Frontend: Vanilla HTML/CSS/JS (no framework)
  • Backend: Python FastAPI
  • Database: PostgreSQL on Railway
  • Scraper: BeautifulSoup + deep-translator
  • Hosting: Railway (auto-deploy from GitHub)
  • Auth: HTTP Basic Auth
Update Schedule

The scraper runs automatically every day at 6:00 AM UTC (1 AM Eastern, 10 PM Pacific).

It checks for new listings, stops when it hits 5 consecutive items already in the database. New items appear automatically - no manual intervention needed.

Known Limitations
  • ~2,000 older items (pre-2016) not yet scraped due to initial connection timeout
  • Some Japanese characters display incorrectly in titles
  • Translation quality varies - some descriptions are awkward
  • Measurements extracted via regex - may miss unusual formats
  • Images hosted on Mushroom's server - may break if they change URLs
